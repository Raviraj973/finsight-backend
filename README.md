# FinSight Backend

AI-powered financial backend system built using Java, Spring Boot, REST APIs, and MySQL.

## Features

* Secure user authentication
* Email verification using UUID activation
* RESTful API architecture
* AI-powered financial insights
* OpenRouter API integration
* Finnhub market data integration
* Dynamic email support using Brevo API
* Conditional API routing
* Layered backend architecture

## Tech Stack

* Java
* Spring Boot
* MySQL
* Hibernate / JPA
* REST APIs
* Maven

## APIs Used

* OpenRouter API
* Finnhub API
* Brevo Email API

## Project Structure

* Controller Layer
* Service Layer
* Repository Layer

## Learning Outcomes

* Backend development
* API integration
* Authentication systems
* Database management
* Layered architecture
* REST API design
